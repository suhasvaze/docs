# 1. What is General Ledger

# 1. Journals Repository

GL is an Application/Module that performs mainly the Accounting and Financial Reporting function (and more...)

  1. GL is Repository of Accounting Journals (Journal Entries as per traditional accounting terminology)

  2. Journals can come in GL from multiple sources View Diagram
    (a) Manually Entered
    (b) Generated from seeded1 Subledgers (SLA - Subledger Accounting)
    (c) Imported from External Systems (AHC - Accounting Hub Cloud)
    (d) Imported from Spreadsheets (FBDI - File Based Data Import)
    (e) Imported from GL Interface

1 Examples of seeded Subledgers are AP, AR, FA, Cash, Inventory, WIP etc
